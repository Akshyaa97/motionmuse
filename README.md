**Step-by-Step Guide:**

**Download the Plugin:**
Click the green <> Code button at the top of this GitHub page and select Download ZIP.

**Extract the Folder:**
Unzip the downloaded file to a permanent folder on your computer. (Don't delete this folder later, as Figma will read the plugin directly from here!)

**Open the Figma Desktop App:**
Launch Figma on your computer and open any design file.

**Open the Development Menu:**
Click the Figma logo (dropdown menu) in the top-left corner.
Navigate to Plugins > Development > Import plugin from manifest...

**Select the Manifest File:**
A file browser will pop up. Locate the unzipped folder from Step 2.
Inside that folder, select the manifest.json file and click Open.

**Run the Plugin:**
You're all set! To use it, simply right-click anywhere on your Figma canvas, go to Plugins > Development, and click the plugin name to launch it.

**Pro-tip:** You can also launch it quickly by using Figma's Quick Actions menu (Cmd + / on Mac or Ctrl + / on Windows) and typing the plugin's name!
